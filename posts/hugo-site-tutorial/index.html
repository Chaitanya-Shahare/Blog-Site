Hugo Site Tutorial

Posted by Chaitanya Shahare on Sun, Jan 22, 2023

Create Hugo Site

In a directory, run the following command

hugo new site porfolio

This will create a new folder named portfolio, which will be our site’s directory.

Get theme

cd porfolio

git clone https://github.com/charlola/hugo-theme-charlolamode.git themes/charlolamode

Config.toml

now paste the following in porfolio/config.toml

baseURL = 'http://example.org/'
languageCode = 'en-us'
title = 'Chaitanya Shahare'



# title = "charlola.mode"
paginate = 5
theme = "charlolamode"
enableInlineShortcodes = true
enableRobotsTXT = true
enableEmoji = true
pygmentsUseClasses = true

[[menu.main]]
name = "Contact"
url = "/contact/"
weight = 30

# [[menu.main]]
# name = "Articles"
# url = "/articles/"
# weight = 20

[[menu.main]]
name = "Projects"
url = "/projects/"
weight = 15

[params]
author = "Author's Name"
defaultTheme = "auto"
ShowShareButtons = true
ShowReadingTime = true
displayFullLangName = true
ShowBreadCrumbs = true
ShowCodeCopyButtons = true
ShowToc = true
mainSections = [ "articles" ]

Live server

In a new terminal, navigate to the portfolio directory

Then run the following command

hugo server

You should be able to see live server
